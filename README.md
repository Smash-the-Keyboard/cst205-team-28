## Installation
`pip install -r requirements.txt`

## How to Run
- Windows:
```
$env:FLASK_APP = "app.py"
$env:FLASK_DEBUG = "1"
flask run
```
- Mac/Unix:
```
export FLASK_APP=app.py
export FLASK_DEBUG=1
flask run
```

The site will be hosted at http://127.0.0.1:5000/
